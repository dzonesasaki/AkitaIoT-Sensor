<Div Align="right"> 2020.10/15-16 , 20-21 </Div>

## はじめに
IoTなど先進的な情報通信技術の導入は、人手不足の問題解消や働き方改革に向けた取り組みとして、重要となっています。\\
本研修では、複数のセンサを用い、必要に応じたセンサ選定とそのセンサを用いたセンサノードの製作手法を学びます。
